ARCHITECTURE

Our Application provides user with a functionality where user can register to use our application to use various blockchain related services with ease. Architecture is designed in such a way that user stays away from the legalities of smart contract and database configuration and provides user a single app review off all the applications.

User needs to have metamask account in order to use any of the application functionality at it's fullest. Metamask is blockchain currency wallet. Data of user is securely stored is decentralised database where user is the only responsible person fpr his data. He/She can never alter the changes made.
